# Simulador Ecommerce - JavaScript

## Descripción

Este proyecto consiste en un simulador interactivo de compras tipo Ecommerce, desarrollado con HTML, CSS y JavaScript. Permite al usuario visualizar productos, agregarlos a un carrito, calcular el total y gestionar la compra de manera dinámica.

## Funcionalidades

* Visualización dinámica de productos desde un archivo JSON
* Agregar productos al carrito
* Incrementar cantidad de productos
* Eliminar productos del carrito
* Cálculo automático del total
* Persistencia de datos mediante localStorage
* Notificaciones mediante la librería Toastify

## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript (ES6)
* Fetch API (asincronismo)
* LocalStorage
* Toastify

## Uso

1. Clonar el repositorio:

git clone https://github.com/Alv-brz/Proyecto-de-JS.git

2. Abrir el proyecto con Live Server o en el navegador

3. Interactuar con los productos y el carrito
